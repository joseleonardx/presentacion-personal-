# JM Solution - Landing Page

Landing page creada con **React** y **Vite** para JM Solution, una propuesta de servicios de soluciones en casa dirigida a amas de casa y personas del hogar.

## 🚀 Descripción
JM Solution ofrece atención rápida y confiable para problemas domésticos, desde reparaciones y mantenimiento hasta mejoras en el hogar. El sitio permite agendar una cita y enviar la descripción del problema directamente por correo.

## ✨ Características del proyecto
- Diseño moderno, oscuro y minimalista.
- Layout responsivo para móviles y escritorio.
- Componentes separados: `Header`, `Hero`, `FeatureCard`, `Footer`.
- Cards renderizadas con `Array.map()` y `key` en cada elemento.
- Formulario de contacto que genera un correo con los datos del usuario.

## 🛠️ Tecnologías utilizadas
- React
- Vite
- JavaScript (JSX)
- CSS propio
